DuoCode

DuoCode is a code learning app that is inspired by Duolingo. It makes learning code more fun and accessible than ever. It caters to a diverse range of skill levels and needs. Start learning today!

How-to

Sign up or Sign in to get started! DuoCode is accessible via the web and can be used on any device with a browser. Signing in will redirect to a homepage containing the list of units. Each unit contains three exercises ordered by difficulty. Click on a blue star icon to try one out!

Each exercise has a prompt that includes a technical coding question. There are three of these questions per exercise. Try your best!

Finally, there is a milestone quiz at the end of the three exercises. The milestone contains three random short response questions from that unit. Complete it to finish the unit. No pressure!
