DuoCode

DuoCode is a code learning app that is inspired by Duolingo. It makes learning code more fun and accessible than ever. It caters to a diverse range of skill levels and needs. Start learning today!

How-to

Sign up or Sign in to get started! DuoCode is accessible via the web and can be used on any device with a browser.

Issues

Found a bug?! Please make an issue in the public Git repository using the bug report template.

Missing Functionality

The following features intended for final release are not complete:
  1. User progress is not tracked.
  2. Scaling is incomplete for small screens.
  3. Styling is incomplete.
  4. Lesson completion logic is missing.
